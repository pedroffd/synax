# Image Enhancer 📸

Technical guide for enhancing image quality for professional use.

## Core Capabilities
- **Resizing & Scaling**: Upscale images while maintaining fidelity using best-fit algorithms.
- **Sharpness & Clarity**: Reduce blur and enhance edges for cleaner documentation.
- **Color Correction**: Balance brightness and saturation for visual impact.
- **Denoising**: Remove artifacts from low-quality screenshots.

## Best Practices
1. **Preserve Aspect Ratio**: Never distort dimensions unless explicitly requested.
2. **Metadata Hygiene**: Strip sensitive EXIF data before final delivery if requested.
3. **Format Optimization**: Default to WebP or AVIF for the web, PNG for documentation.
4. **Consistency**: Ensure all images in a collection share similar tonal values.

## When to Use
- Before adding screenshots to `walkthrough.md`.
- Preparing assets for an implementation plan.
- Creating "Wow" visuals for stakeholder presentations.

**Golden Rule**: Enhancement should look natural. Avoid over-sharpening or artificial color saturation.
